# Changelog

## 2.0.0-beta.5

- New module-first standalone engine.
- Strict pack and module SHA-256 verification.
- Declarative blocks, items, recipes, and entities.
- Trusted executable module lifecycle and data-only addons.
- Async chunk generation and meshing.
- Dependency-free software voxel renderer.
- Player physics, mining, placement, inventory, crafting, survival, entities, and combat.
- Atomic save metadata and resource-ID chunk palettes.
- Ashfall Standalone Engine Edition graph-first playable verification slice.
- Headless, render, module graph, JSON, and save compatibility tests.
